# Security

## Reporting a vulnerability

If you believe you have found a security vulnerability, please do not open a public issue.

- Preferred channel: open a private report via [GitHub Security Advisories](https://github.com/ArthurKeen/arango-schema-mapper/security/advisories/new).
- Include: a description, reproduction steps, affected versions, and impact assessment.
- We aim to acknowledge reports within 7 calendar days.

## Secrets handling

This project **must not** persist API keys or credentials in logs, cache outputs, or generated artifacts.
